・はじめに
TwoFacedをダウンロードして頂きありがとうございます．
システムを組み上げたプロトタイプ版になります．
実装されているステージは1つ実装されています．
攻撃モードと守備モードを上手く切り替えて戦いクリアを目指してみてください．


・操作方法
キーボード/Xinputコントローラー
決定・射撃 -> Space,↓キー/Aボタン
モード切替 -> V，→キー/Bボタン
ポーズ -> Homeキー/スタートボタン
ライセンス表示 -> F1キー

・ステータス説明
「攻」「守」-> 現在のモードを表示 攻だと前方へ弾を発射，守だと前面に壁を生成して弾を防ぎます
「体」-> 体力です．ゲージが一番下まで下がるとゲームオーバーです．
「力」-> 弾，壁を作成する際に必要な力です．一番下まで下がると一番上に上がるまで弾も壁も出せません．
「待」-> モードを切り替えた際に次に切り替えられるまでの待ち時間です．一番上までゲージが上がると切り替えられます．
